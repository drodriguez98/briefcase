SPRING BOOT & ANGULAR PROJECTS

Los siguientes dos proyectos comparten similitudes al estar desarrollados con Java utilizando Spring Boot para el backend y Angular o Thymeleaf para el frontend, ofreciendo diversas funcionalidades y permitiendo a los usuarios interactuar de manera diversa.

Estos proyectos han brindado la oportunidad de adquirir y fortalecer habilidades clave, tales como:

Con Angular, he podido aprovechar herramientas como Angular Material y ngx-charts para mostrar datos de manera atractiva y eficiente, mientras que Thymeleaf me ha permitido generar plantillas de forma rápida y efectiva para el frontend.
  • Manipulación y visualización de datos: El manejo de datos y su visualización a través de gráficos o tablas, mediante el uso de herramientas como angular-material o ngx-charts, ha ampliado mi conocimiento en la presentación de información de manera efectiva y comprensible para los usuarios.

PYTHON

Los siguientes tres proyectos comparten similitudes, ya que todos están desarrollados con la biblioteca Python Pygame y comparten ciertos elementos comunes en su estructura y funcionalidad.

Para desarrollarlos, he utilizado las siguientes tecnologías y técnicas.

  • Python y Pygame. El juego está escrito en Python y utiliza la biblioteca Pygame para manejar la interfaz gráfica, eventos de teclado, sonidos y animaciones.
  • Manipulación de gráficos. Pygame proporciona funciones para dibujar gráficos en una ventana, como la creación de rectángulos de colores, animaciones y actualización de la pantalla.
  • Eventos de usuario. El juego responde a eventos de usuario, como clics de ratón y pulsaciones de teclas, para permitir la interacción del jugador.
  • Sonidos y música. Pygame se utiliza para reproducir sonidos y música cuando el jugador realiza acciones específicas, como coincidir con un patrón correcto o perder el juego.
  • Control de tiempo. El juego utiliza el módulo de tiempo para gestionar el tiempo, controlar la velocidad de las animaciones y establecer límites de tiempo para ciertas acciones, como el tiempo antes de que se considere que el jugador ha perdido.
