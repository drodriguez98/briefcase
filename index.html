Desarrollador con experiencia en una amplia gama de tecnologías. Desde bases de datos como SQL hasta lenguajes de programación como Python, PHP, y Java, incluyendo el desarrollo con Spring Boot. Además, tengo experiencia en herramientas de front-end como HTML5, CSS3, Javascript, y el Framework Angular.

Mi formación incluye una licenciatura en Educación Infantil, un Ciclo de Grado Superior en Administración de Sistemas en Red y un Bootcamp de Programación. Estas experiencias me han proporcionado una perspectiva única y un conjunto diverso de habilidades que aplico en cada proyecto.

A pesar de que mi carrera profesional está en sus primeras etapas, mi enfoque autodidacta y mi capacidad para adaptarme rápidamente a nuevas tecnologías me han permitido crear soluciones innovadoras y sólidas. Mis proyectos personales reflejan mi habilidad para desarrollar soluciones creativas y funcionales. He enfrentado desafíos diversos y he logrado soluciones efectivas que demuestran mi pasión y competencia como desarrollador.

Mi objetivo es seguir creciendo como profesional, contribuir con mi experiencia y creatividad a proyectos desafiantes, y continuar aprendiendo y mejorando mis habilidades en el campo del desarrollo de software y/o web.

SPRING BOOT & ANGULAR PROJECTS

The following three projects share similarities, as they are all developed with the Pygame Python library and share certain common elements in their structure and functionality. Below I provide an overall summary highlighting the specific settings that differentiate each of them.

To develop them I have used the following technologies and techniques.

  • Python and Pygame: The game is written in Python and uses the Pygame library to handle the graphical interface, keyboard events, sounds and animations.
  • Graphics Manipulation: Pygame provides functions for drawing graphics in a window, such as creating colored rectangles, animations, and updating the screen.
  • User Events: The game responds to user events, such as mouse clicks and keystrokes, to allow player interaction.
  • Sounds and Music: Pygame is used to play sounds and music when the player performs specific actions, such as matching a correct pattern or losing the game.
  • Time Control: The game uses the time module to manage time, control the speed of animations and set time limits for certain actions, such as the time before the player is considered to have lost.
